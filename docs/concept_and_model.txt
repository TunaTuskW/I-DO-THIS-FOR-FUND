MACRO ANALYST AGENT - CONCEPT AND MODEL
=======================================

1. WHAT IT IS
This is a 24/7 autonomous quantitative agent that monitors global markets, processes data through deep learning classifiers, and generates institutional-grade macro reports. It runs completely automatically in the background using your computer's scheduler. The presentation layer uses an ultra-minimal "Brutalist Terminal" text-block layout to ensure a data-dense, cross-platform reading experience designed for instant mobile parsing.

2. THE FULLY-PYTHON DETERMINISTIC ARCHITECTURE
Following the v4.0.0 upgrade, the agent is completely Python-driven and 100% deterministic. The system no longer requires feeding a "wall of text" prompt into a Large Language Model (LLM). Instead, the architecture relies on mathematical consensus logic. 

- Data Ingestion & Signal Layer: Real-time data is ingested via Yahoo Finance (equities, commodities, crypto flow, VVIX) and the FRED API (yields). An NLP NewsSignalVector ingests RSS feeds, interpreting sentiment purely as an absolute shock/volatility multiplier rather than directional bias.
- Core Quantitative Engines: The Python pipeline tracks "Regime Persistence" (how long the market has spent in its current state) and a "Hidden Fragility Index" (monitoring VVIX/VIX expansion and SPX/DXY correlation).
- The Consensus Engine: The script aggregates 5 distinct quantitative signals (Kalman State, MCS, Volume Heat, Kelly Size, Extremes) into `ModelResult` dataclasses and runs a conviction-weighted vote. If the models diverge wildly, the Shannon Entropy forces an automatic "NO PREDICT / STAND ASIDE" posture.

3. THE END-TO-END WORKFLOW
- Data Collection: Every cycle, the python scripts download the massive data payload.
- Mathematical Scoring: The python scripts score the market stress and momentum.
- Automated Formatting: The Python scripts natively format the mathematical output into the Brutalist markdown layout.
- Delivery: The final report is instantly pushed to your Discord channel.
- Cryptographic Proof: Every report is digitally signed and "hash-chained" (TruChain) so that its history cannot be secretly altered later.

4. THE MATH & MODELS BEHIND THE SCENES
- Macro Condition Score (MCS): A composite number measuring overall market stress based on stock momentum, interest rates, energy prices, and cross-asset behavior.
- Hidden Markov Models (HMM) & Kalman Filters: These algorithms smooth out daily noise to figure out the "true" underlying regime (Risk-On, Risk-Off, or Transitional).
- Deep Classifier (MLP): A trained neural network that double-checks the HMM. 
- Institutional Footprinting: Natively supports advanced institutional footprinting, including a 20-day continuous Institutional Heat Index (IHI).

5. ADVANCED MODEL GOVERNANCE & TACTICAL ASSUMPTIONS
- Kelly Sizing Decay Penalty: The algorithm targets exposure using the Fractional Kelly Criterion, heavily penalized by historical Brier Score calibration. Furthermore, if a regime outlives its mathematical "half-life" (e.g., 11 days for RISK_ON), an exponential decay penalty forcefully scales the portfolio exposure down, enforcing systematic profit-taking before transitions occur.
- 1.5 Std Dev Dynamic Boundaries: Continuously tracks volatility thresholds to automatically invalidate outdated narratives upon statistical sweeps.

6. ESCALATION SYSTEM
The agent respects your time and won't spam you with panicky alerts. It only flags urgency when the math demands it:
- ROUTINE: Normal market updates.
- ELEVATED: Moderate volatility spikes, sudden currency squeezes, or early-warning transition signals.
- CRITICAL: Extreme mathematical stress (e.g., VIX > 30, simultaneous stock and bond crashes). Discord will ping you heavily for these.

7. THE AUTOMATED SCHEDULE
The agent manages three distinct workflows natively via bash scripts (run_4h.sh, run_daily.sh, run_weekly.sh) executed by cron:
- The 4-Hour Briefing: A quick pulse-check of the active market session.
- The Daily Roll: At midnight, it pushes a master document containing the last 72 hours of updates.
- The Sunday Synthesis: A deep-dive research report summarizing the entire week and previewing the week ahead.

8. OFFLINE MODEL TRAINING & BACKTESTING
The deep learning models (HMM and MLP) are not static. The agent includes an offline training pipeline (train_models.py) that allows you to periodically re-train the neural networks on fresh market data. Whenever you train the models, it automatically generates a Backtest Report (backtest_results.md) so you can empirically verify the agent's mathematical edge before it trades or analyzes live markets.

By anchoring the system to a rigid mathematical backbone, the pipeline provides institutional-quality insights with deterministic reliability, entirely free from LLM hallucinations.
